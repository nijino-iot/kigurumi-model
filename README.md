# Kigurumi 头壳模型开源项目

欢迎来到 Kigurumi 头壳模型开源项目！本项目致力于分享和改进 Kigurumi（着ぐるみ）头壳的 3D 模型设计。

定制头壳可以加QQ群：1036511798

淘宝店：[MEIS莓萌KIGURUMI](https://meis.taobao.com)

3. **必须开源**：如果您基于本项目进行了修改或二次创作，您必须：
